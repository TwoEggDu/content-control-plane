## 使用规则

- 新工作优先进入 backlog，而不是直接塞进 README
- 真正影响方向的事项才进入 `owner_decisions`
- 已完成事项可以保留一段时间，避免阶段信息丢失
- 如果 backlog 长期失真，应优先修 backlog，而不是继续加文档
