# 待你决策

这个目录用于集中管理所有需要你拍板的事项。

## 目录规则

- 每天新增的事项，写入当天文件，例如 `2026-03-11.md`
- 当前仍未决的事项，统一汇总到 `current.md`
- 作为 owner 平时需要留意什么，单独维护在 `watchlist.md`
- 已经决策的事项，后续可以从 `current.md` 移出，并保留在对应日期文件中作为历史记录

## 你优先看什么

优先看这两个文件：

- `current.md`：当前仍待你决定的事项总表
- `watchlist.md`：当前阶段你最该盯的东西

如果你要补当天新增背景，再看日期文件，例如 `2026-03-11.md`。

## 你的回复方式

你可以直接这样回复：

- `D-001：选择推荐方案`
- `D-002：选择 B`
- `D-003：延后`
- `D-004：自定义答案：...`

## 状态值约定

- `OPEN`
- `DECIDED`
- `DEFERRED`
- `OBSOLETE`
